Cliente x Servers

Clietes são as maquinas que recebem as informaçoes, Server sao maquinas que armazenam as informaçoes.

  1. Clientes > Celulares > Tablets > Computadores = INTERNET
  2. Servidores > INTERNET

Para acessar uma página, o cliente deve clicar em um link, que resultará em um comando enviado para a Internet, que por sua vez encaminhará a solicitação a um servidor. Este servidor responderá com um site, e-mail, arquivo, etc.

Por outro lado, um servidor é um sistema de computador dedicado a atender solicitações de clientes. Os servidores armazenam e disponibilizam informações, sites, aplicativos e outros recursos online.

Oque é HTML

HTML, sigla para "Hypertext Markup Language," é uma linguagem de marcação fundamental para a construção de páginas web.

HTML não é uma linguagem de programação; é uma linguagem de marcação. Com ela, podemos estruturar códigos. Assim como na aula, aprendi a enxergar o HTML como se fosse o esqueleto do corpo, e isso ajudou muito na organização dos códigos..

As tags que foram empregadas neste início, desde o "strong" até o "p", são ótimas para desenvolver um corpo na escrita HTML.

Conclusao

Para um início básico, consegui entender bem o funcionamento das tags e o quanto elas abrangem os elementos atribuídos a elas.
